PHP/SQL

Mijn PHP avontuur is begonnen op CodeCadamy. Hier werden duidelijk de beginselen uitgelegd en door middel van oefeningen gelijk toepassen.

Heb een paar voorbeelden onder mijn github gezet. Moeten op een lokale server uitgevoerd worden, ik gebruikte Wampserver64.

Na Codecadamy ben ik beginnen aan Building Database Applications in PHP, aangeboden door Coursera. Doel van de cursus was een volledige CRUD applicatie bouwen. De cursus wordt gegeven door een ervaren programmeerder die goed de redenen gaf waarom PDO goed was en op welke manieren SQL-injectie kon worden tegengegaan. Zie mijn Coursera github voor deze opdracht. Er zit een document bij die de benodigde stappen uitlegt.
